Quick Introduction to Angular

Angular is a framework for building client applications in HTML, CSS and JavaScript/TypeScript. Also it allows you to create reactiveSingle-Page-Applications (SPAs)

Why to use Angular Vs vanilla JavaScript?

Benefits of using Angular:

  • Gives our application a clean structure
  • Includes a lot of re-usable code
  • Makes our applications more testable

The most quickest way to start up with Angular application is with the Angular CLI (Command Line Interface).
