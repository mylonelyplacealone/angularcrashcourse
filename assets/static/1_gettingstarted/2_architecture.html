Architecture of Angular Apps

Front-end

Angular application are front end only and data can not be saved within Angular apps persistently.

Angular represent only user interface which will be built with HTML templates and Presentation logic and technologies such as HTML, CSS, TypeScript, Angular will be used to achieve this.

Back-end

On the other side, Angular application can use different backend technologies for data and processing such as APIs and HTTP requests which will implement business logic.

Backend systems will interact with databases and serves the data get/post requests from Angular front end.
