Property Binding

From the exapmle below, Angular pulls the value from fullName property from the component and inserts it using the html property innerHtml of <h1> element.


import { Component } from '@angular/core';

@Component({
    selector: 'my-app',
    template: `
                <h1 [innerHtml]='fullName'></h1>
              `
})
export class AppComponent {
    fullName: string = 'Martin Luther King Jr';
}

To set an element property to a non-string data value, you must use property binding.
