How to Install Angular 6

The most quickest way to start up with Angular application is with the Angular CLI (Command Line Interface). 

This can be downloaded from the node package manager(npm). To check whether you have npm, open up windows command prompt or your IDE terminal window and type:

> npm -v

If we get some version information then we are good to move to next step. Otherwise if you receive unrecognized (and not a version number), you need to install NodeJS. Once you install NodeJS, you will have access to npm.

Now, npm is ready to install the Angular CLI.

The Angular CLI official link - Angular CLI.

You can follow above official link for the commands to run to setup Angular CLI. Otherwise continue next steps which will run you thorugh same steps.

> npm install -g @angular/cli

If you run ng -v after installing, it will provide you with the version number.

Once the CLI is installed, let's create our first a brand new Angular 6 project with below command:

> ng new [Your Project Name Here] --routing
  • ng - This is how we need to call the Angular CLI.
  • new - This is one of the many commands we can run/execute with the Angular CLI.
  • [Your Project Name Here] = This is the name of our project we want to create. The CLI will create a folder with this name under the folder where you are running this command.
  • --routing - This is the optional flag out of plenty available. This will create a routing file for us inside the project which we will be using for application page routing.

To check out all of the available commands and options, run ng at the command line.

Once the CLI has generated the new project, you can check into it:

> cd [Your Project Name Here]

If you use Visual Studio Code, you can type code . to launch the code editor. Then, to start serving the project to the browser, you need to run:

> ng serve -o

The -o flag tells the CLI to launch your browser with the project. Now, you're able to watch your Angular 6 project as you code and it will be automatically reload on each updates in teh application.
